Section 4: Event-Driven Animation Behaviors

How each bot event triggers character reactions — professional baseline, dramatic on big moments

Three Behavior Tiers

TIER 1: Professional Baseline
Default state. Characters sit at desks, type, check monitors, subtle head movements. Scanner occasionally picks up phone. Risk Manager leans back in chair. This is the "office at work" feel.
TIER 2: Active Trading
When signals fire, positions open/close. Characters stand up, point at screens, lean forward urgently. Executor types fast. Strategy Engine gestures to Ensemble. Energy picks up.
TIER 3: Wolf of Wall Street
Streaks, big wins, big losses, milestones. High-fives, desk slams, team huddles, Jonas standing and clapping, celebration huddle. The emotional peaks.

Big loss (> $2) T3 Executor, Risk Manager Executor slams desk (brief frustrated gesture). Risk Manager stands up, walks to Executor's desk for review.

Drawdown > 10% T3 ALL Tension mode. Everyone stops casual animations. Head-in-hands for some. Jonas pacing behind desks. Risk Manager standing at his station with arms crossed.
New balance ATH T3 ALL Big celebration! Jonas stands on desk (or raises arms). Team erupts — clapping, high-fives, pointing at the main display showing the new peak.

Visual Status Indicators (Replacing Plumbobs)

Plumbobs are Sims-specific. For realistic characters, status is shown through:
Monitor Glow
Each desk monitor reflects agent status — green (active), amber (waiting), red (alert), blue (scanning)
Desk LED Strip
Subtle LED under desk edge — changes color with agent state (like gaming desk RGB but tasteful)
Floating Name + Role
Clean CSS2D label above each character — name, role, current status text (replaces emoji)
