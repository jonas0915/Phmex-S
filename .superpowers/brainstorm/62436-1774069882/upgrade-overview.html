Design Section 1: What Changes vs. What Stays

The upgrade preserves your scene structure and swaps the rendering quality

STAYS THE SAME (Structure)

  • Scene layout — same desk positions, room dimensions (12×10×4), stairwell, conference room, break room
  • Camera — same isometric angle (2, 6, 9), OrbitControls, double-click reset
  • 8 agents + Jonas — same positions, same roles, same desk assignments
  • Event system — coffee breaks, meetings, facility visits, team events (same intervals)
  • HUD layout — bottom status bar, right comms panel, left intel panel
  • Data pipeline — reads trading_state.json + bot.log only, zero API calls
  • Sleep system — 11pm-6am behavior
  • Lower floors — gym, cafeteria, rec room, bar, bedrooms, jacuzzi
  • Dialogue/speech bubble system

UPGRADES (Visual Quality)

Component Current (Sims) Upgraded (COD-quality)
Characters Procedural spheres/cylinders, chunky Sims proportions, ~50 meshes each GLTF rigged humanoid models from Mixamo, realistic proportions, PBR skin/clothing materials, bone-based animation
Animations Manual sine-wave arm/leg rotation, basic walk cycle Mixamo motion-capture animations: idle-seated, typing, walking, standing-up, celebrating, head-in-hands, pointing
Faces Sphere head + sphere eyes/nose/lips, painted-on features Detailed mesh faces with proper topology, blend shapes for expressions (smile on wins, frown on losses)
Hair Capsule/sphere combos (swept, crew, bun, etc.) Mesh-based hair geometry baked into GLTF models, proper shading
Clothing Colored cylinder/capsule body parts Textured clothing meshes — suits, hoodies, casual wear with fabric materials
Lighting 7 lights, 1024×1024 shadow map, basic ACESFilmic Same light count but upgraded: 2048×2048 shadows, SSAO (ambient occlusion), contact shadows, improved bloom
SF Bay Panorama Procedural canvas (2048×768), flat building silhouettes HDRI-quality panoramic skybox (4096+ res), detailed SF landmarks (Golden Gate, Transamerica, Salesforce Tower), realistic water reflections
Furniture Procedural boxes/cylinders for desks/chairs GLTF models: proper office furniture with bevels, curves, realistic materials
Post-Processing UnrealBloom + SMAA Bloom + SMAA + SSAO + screen-space reflections + color grading + vignette
Floor/Walls Flat color planes, glass opacity 0.04 Normal-mapped concrete floor, physically-correct glass with reflections, proper mullion detail

NEW ADDITIONS

  • Jonas character — modeled from jonas_avatar.jpg reference, casual tech CEO look (hoodie + jeans), standing desk command station
  • Dramatic event reactions — high-five on win streaks, desk slam on stop losses, team huddle on big decisions
  • Agent remapping — 8 agents mapped to real bot subsystems (Scanner, Risk, Ensemble, Executor, Strategy, Tape, WS Feed, Position Monitor) replacing decorative roles (Therapist → Position Monitor, etc.)
  • Asset loading system — GLTFLoader pipeline for models, textures, animations (~15-20MB total)
  • LOD system — level-of-detail to maintain 30+ FPS
